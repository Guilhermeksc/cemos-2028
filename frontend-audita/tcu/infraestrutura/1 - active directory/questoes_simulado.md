Questão 1

Ano: 2025
Banca: Simulado
Tipo: múltipla escolha

Enunciado:
O serviço do Active Directory responsável por armazenar informações sobre usuários, computadores e dispositivos é o:

a) AD CS
b) AD DS
c) AD FS
d) AD RMS
e) AD LDS

Resposta correta: b
Comentário: O Active Directory Domain Services (AD DS) é o serviço que mantém o banco de dados principal com informações sobre objetos da rede.

Questão 2

Ano: 2025
Banca: Simulado
Tipo: múltipla escolha

Enunciado:
Qual dos seguintes serviços do Active Directory é responsável pela criação e gerenciamento de certificados digitais de chaves públicas?

a) AD DS
b) AD FS
c) AD RMS
d) AD CS
e) AD LDS

Resposta correta: d
Comentário: O Active Directory Certificate Services (AD CS) provê infraestrutura de certificação para autenticação e comunicação segura.

Questão 3

Ano: 2025
Banca: Simulado
Tipo: múltipla escolha

Enunciado:
O Active Directory Federation Services (AD FS) tem como principal função:

a) Gerenciar certificados digitais.
b) Armazenar informações sobre domínios.
c) Permitir autenticação única em múltiplas plataformas.
d) Criar partições de esquema.
e) Gerar chaves de criptografia assimétrica.

Resposta correta: c
Comentário: O AD FS permite logon único (SSO) entre sistemas heterogêneos, integrando identidades de diferentes plataformas.

Questão 4

Ano: 2025
Banca: Simulado
Tipo: múltipla escolha

Enunciado:
O serviço do Active Directory que oferece funcionalidades semelhantes ao AD DS, porém sem exigir a criação de domínios ou controladores de domínio é o:

a) AD RMS
b) AD LDS
c) AD FS
d) AD CS
e) AD DS

Resposta correta: b
Comentário: O Active Directory Lightweight Directory Services (AD LDS) é uma versão leve do AD DS usada para aplicações que não precisam de um domínio completo.

Questão 5

Ano: 2025
Banca: Simulado
Tipo: múltipla escolha

Enunciado:
O AD RMS (Rights Management Services) é utilizado para:

a) Replicar objetos entre controladores de domínio.
b) Gerenciar identidades federadas.
c) Armazenar dados de diretório.
d) Controlar o uso e a proteção de informações.
e) Emitir certificados digitais.

Resposta correta: d
Comentário: O AD RMS fornece proteção de informações, controlando como documentos e e-mails podem ser utilizados.
