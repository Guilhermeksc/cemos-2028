| Princípios de Relações Internacionais | Cap 4 | Sistema Internacional | Para os liberais, o sistema internacional é caracterizado por **múltiplos atores**, incluindo organizações internacionais, ONGs e corporações multinacionais, além dos Estados. | Para os liberais, assim como para os realistas, o Estado é o único ator relevante no sistema internacional, e os demais são meramente ferramentas estatais. | true | Os liberais veem o sistema como interdependente, com múltiplos atores interagindo (Estados, IGOs, ONGs, corporações multinacionais, etc.). | | |

| Princípios de Relações Internacionais | Cap 4 | Sistema Internacional | O Radicalismo descreve a estrutura do sistema internacional em termos de **estratificação**, que é a distribuição desigual de recursos entre diferentes grupos de Estados. | O Radicalismo descreve a estrutura do sistema internacional primariamente pela polaridade militar, ignorando a distribuição desigual de recursos econômicos. | true | A estratificação (distribuição desigual de recursos) é o conceito-chave para entender o sistema internacional sob a ótica radical. | | |
| Princípios de Relações Internacionais | Cap 4 | Sistema Internacional | Para os radicais de base marxista, o capitalismo é o principal responsável pela estratificação, gerando instrumentos de dominação, como as corporações multinacionais, que fortalecem o Norte. | Os radicais afirmam que a estratificação é causada unicamente por diferenças culturais e religiosas entre o Norte e o Sul, sem qualquer ligação com o sistema capitalista. | true | O capitalismo introduz e perpetua a estratificação, estruturando a relação entre os privilegiados (Norte) e os desprivilegiados (Sul). | | |
| Princípios de Relações Internacionais | Cap 4 | Sistema Internacional | A instabilidade no sistema estratificado radical surge quando as **potências situadas na segunda divisão** desafiam as potências dominantes, buscando reverter seu status. | A instabilidade no sistema radical é sempre causada pelas potências da primeira divisão, que atacam Estados pequenos e periféricos para manter a hegemonia. | true | As potências desafiadoras (segunda divisão) podem causar instabilidade ao buscar ascender e obter recursos, como a Alemanha na década de 1930. | | |

| Princípios de Relações Internacionais | Cap 4 | Sistema Internacional | Para os realistas, a mudança fundamental no sistema internacional é frequentemente causada por **guerras**, que resultam em uma redistribuição de poder entre os atores principais. | Os realistas defendem que a mudança no sistema ocorre quase exclusivamente devido à evolução paulatina das normas sociais e à internalização de novas ideias pelos Estados. | true | As guerras são frequentemente as responsáveis por mudanças nas relações de poder, levando a transformações no sistema internacional (ex: Fim da Segunda Guerra Mundial). | | |
| Princípios de Relações Internacionais | Cap 4 | Sistema Internacional | O Liberalismo enxerga a mudança no sistema internacional como um processo **paulatino e constante**, resultante de desenvolvimentos tecnológicos, novos atores e o aumento da importância de questões transnacionais. | O Liberalismo, assim como o Radicalismo, defende a realização de profundas e imediatas modificações estruturais para alterar o sistema internacional. | true | A mudança para os liberais é paulatina e constante, vindo de desenvolvimentos tecnológicos, novos atores e o aumento da importância de outras áreas além da segurança. | | |
| Princípios de Relações Internacionais | Cap 4 | Sistema Internacional | O conceito de **segurança coletiva**, defendido pelos liberais, baseia-se na ideia de que a paz é indivisível: um ataque contra um Estado é um ataque contra todos. | A segurança coletiva é um conceito realista que postula que cada Estado deve zelar apenas pela sua própria segurança, sem depender de alianças mútuas. | true | A segurança coletiva é uma abordagem liberal baseada na ideia de que a guerra contra um é contra todos, obrigando a comunidade internacional a responder. | | |
| Princípios de Relações Internacionais | Cap 4 | Sistema Internacional | A Teoria da Estabilidade Hegemônica postula que a **unipolaridade** leva ao sistema internacional mais estável, desde que o ator hegemônico esteja disposto a impor as normas. | A Teoria da Estabilidade Hegemônica argumenta que a multipolaridade com equilíbrio de poder é o sistema mais estável e que a unipolaridade é inerentemente volátil. | true | Segundo a Teoria da Estabilidade Hegemônica, a unipolaridade é o sistema mais estável quando o hegemon se dispõe a arcar com o ônus da imposição das normas. | | |
